# 도망프로젝트 (runaway-gift)

화이트데이 컨셉의 인터랙티브 웹 페이지입니다.  
`예` 버튼은 마우스를 피해 화면 전체를 도망가고, `아니오` 버튼은 정상 선택 가능합니다.

## 주요 기능
- 화면 전체에서 도망가는 `예` 버튼
- `예` 버튼 클릭 불가 처리 (`pointer-events: none`)
- 회피 시 랜덤 토스트 문구 표시
  - `잡히기 직전에 도망!`
  - `아슬아슬했어요!`
  - `또 놓쳤네요!`
- 회피 성공 횟수 카운터
- `아니오` 버튼 선택 시 완료 상태 + 컨페티 효과

## 커스터마이징 포인트
- 문구 변경: `index.html`
- 색상/폰트/애니메이션: `style.css`
- 도망 강도/속도/민감도: `app.js`
